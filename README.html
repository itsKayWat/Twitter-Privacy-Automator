Twitter Privacy Automator

Purpose

This tool automates the process of changing Twitter post privacy settings to "Only you" for reply permissions. It's designed for users who want to bulk-modify their tweet privacy settings without manually clicking through each post.

Use Cases

  • Content creators wanting to limit interactions on old posts
  • Users experiencing harassment needing to quickly restrict replies
  • Privacy-conscious individuals wanting to limit engagement on historical tweets
  • Account cleanup and privacy enhancement

Requirements

  • Python 3.7+
  • Chrome Browser
  • Active Twitter account
  • Chrome logged into Twitter

Installation

  1. Clone the repository
  2. Run: python requirements.py
  3. Update the Chrome profile path in the script if needed

Usage

  1. Run: python Twitter-private.py
  2. Enter your Twitter profile URL when prompted
  3. Let the script run - it will automatically process your tweets
